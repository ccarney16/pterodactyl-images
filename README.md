This repo provides a set of images for the Pterodactyl Game Panel. The following images are provided:


OpenJDK:

- Java 17
    - jdk17-corretto
    - jdk17-temurin
- Java 19
    - jdk19-corretto
    - jdk19-temurin
- Java 20
    - jdk20-temurin
- Java 21
    - jdk21-temurin
- Custom images
    - jdk11-deployable
    - jdk17-deployable

GraalVM:

- ce-jdk8
- ce-jdk11
- ce-jdk17
- ce-jdk21
- ce-jdk23

*Note: These images will be based on the latest possible version of Oracle Linux*
